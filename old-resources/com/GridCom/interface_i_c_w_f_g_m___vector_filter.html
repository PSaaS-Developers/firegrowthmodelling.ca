Detailed Description

Interface ICWFGM VectorFilter.

The purpose of this object is to provide static vector fuel breaks (polylines) to fires during a simulation. It is included in the GridCom DLL for simplicity only. It does not support a GridEngine interface, but uses such an object to do bounds and validity checking. It does support a VectorEngine interface for attachment to a scenario.

This object also implements the standard COM IPersistStream, IPersistStreamInit, and IPersistStorage interfaces, for use in loading and saving. Serialization methods declared in these interfaces save all the polyline data.
